# Beispiel-Workflows

## Workflow 1: Jobcenter-Brief

Der Nutzer fotografiert einen Brief vom Jobcenter.

DocPilot erkennt:

- Dokumenttyp: Aufforderung zur Mitwirkung
- Organisation: Jobcenter Musterstadt
- Frist: 15.05.2026
- Handlung: Kontoauszüge der letzten drei Monate einreichen
- Risiko: mögliche Leistungskürzung bei Nichtreaktion
- Fall: Jobcenter 2026

Automatisch erstellt:

- Aufgabe: Kontoauszüge prüfen und hochladen
- Erinnerung: 10.05.2026
- Antwortvorschlag: höfliche Mitteilung mit Anlagenliste

## Workflow 2: Mahnung

Der Nutzer lädt eine Mahnung hoch.

DocPilot erkennt:

- Dokumenttyp: Mahnung
- Betrag: 148,90 EUR
- Zahlungsfrist: 15.05.2026
- Absender: Stadtwerke Musterstadt
- Risiko: weitere Mahngebühren oder Inkasso

Empfohlene Aktionen:

1. Prüfen, ob die Rechnung bereits bezahlt wurde.
2. Kontoauszug im Fall suchen.
3. Bei Zahlung Nachweis senden.
4. Bei Nichtzahlung Zahlung oder Ratenzahlung vorbereiten.

## Workflow 3: Krankenkassen-Ablehnung

Der Nutzer scannt eine Ablehnung der Krankenkasse.

DocPilot erkennt:

- Dokumenttyp: Ablehnungsbescheid
- Frist: Widerspruch innerhalb eines Monats
- Thema: Kostenübernahme
- Risiko: Anspruch kann verfallen, wenn kein Widerspruch eingelegt wird

Automatisch erstellt:

- Fall: Krankenkasse Kostenübernahme
- Aufgabe: Widerspruch prüfen
- Antwortvorschlag: formeller Widerspruch mit Bitte um Akteneinsicht oder Begründung

## Workflow 4: Mietstreit

Der Nutzer hat mehrere Briefe zur Wohnung.

DocPilot kann beantworten:

- Welche Schreiben kamen von der Vermieterin?
- Welche Fristen laufen?
- Welche Mängel wurden schon gemeldet?
- Wann wurde zuletzt geantwortet?
- Welche Nachweise fehlen?

DocPilot erzeugt:

- Fallchronologie
- Fristenliste
- Antwortentwurf
- Dokumentenübersicht
