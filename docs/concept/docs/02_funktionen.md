# Funktionsumfang

## 1. Dokumente erfassen

Nutzer können Dokumente importieren als:

- Foto
- PDF
- JPEG
- PNG
- Scan
- später optional: E-Mail-Anhang

Unterstützte Dokumentarten:

- Behördenbriefe
- Rechnungen
- Mahnungen
- Verträge
- Versicherungsunterlagen
- Arztbriefe
- Befunde
- Krankenkassenunterlagen
- Kontoauszüge
- Anträge
- Kündigungen
- Bescheide
- Gerichtsschreiben
- Steuerunterlagen
- persönliche Korrespondenz

## 2. OCR

Die OCR extrahiert Text aus Bildern und PDFs. Für gute Ergebnisse sollte das System unterstützen:

- automatische Bildkorrektur
- Rotationserkennung
- Kontrastoptimierung
- Mehrseiten-PDFs
- Trennung zwischen OCR-Text und Originaldatei

## 3. KI-Analyse

Die KI erzeugt aus jedem Dokument eine strukturierte Analyse:

- Kurzbeschreibung
- vollständige Zusammenfassung
- erkannter Dokumenttyp
- Sender und Empfänger
- relevante Personen
- relevante Organisationen
- Datum des Schreibens
- Eingangsdatum
- Aktenzeichen
- Kundennummern
- Beträge
- Fristen
- Risiken
- empfohlene Aktion

## 4. Personen- und Organisationsordner

Jede erkannte Person oder Organisation kann einen eigenen Ordner erhalten.

Beispiele:

- Jobcenter Musterstadt
- Krankenkasse
- Finanzamt
- Vermieterin Frau Schneider
- Dr. Müller
- Versicherung XY

Jeder Ordner enthält:

- Dokumente
- Notizen
- erkannte Fristen
- offene Aufgaben
- Verlauf
- Vektorindex für semantische Suche

## 5. Fallverwaltung

Ein Fall bündelt alle Informationen zu einem Thema.

Ein Fall enthält:

- Titel
- Beschreibung
- Status
- Priorität
- beteiligte Personen
- beteiligte Organisationen
- Dokumente
- Fristen
- Aufgaben
- Notizen
- KI-Zusammenfassung
- Chronologie
- nächste empfohlene Schritte

Mögliche Statuswerte:

- Offen
- In Bearbeitung
- Wartet auf Antwort
- Frist läuft
- Kritisch
- Erledigt
- Archiviert

## 6. Fristenerkennung

DocPilot erkennt Formulierungen wie:

- „Bitte zahlen Sie bis zum 15.05.2026.“
- „Widerspruch innerhalb eines Monats nach Zugang.“
- „Unterlagen innerhalb von 14 Tagen einreichen.“
- „Termin am 12.06.2026 um 10:30 Uhr.“

Fristtypen:

- Zahlungsfrist
- Widerspruchsfrist
- Einsendefrist
- Gerichtstermin
- Arzttermin
- Kündigungsfrist
- Vertragsfrist
- persönliche Aufgabe

## 7. Aufgaben

Aus Dokumenten entstehen automatisch Aufgaben.

Beispiele:

- Rechnung prüfen
- Betrag überweisen
- Widerspruch vorbereiten
- Unterlagen nachreichen
- Termin wahrnehmen
- Antwort schreiben
- Dokument versenden
- Nachweis suchen

## 8. Benachrichtigungen

Mögliche Erinnerungsregeln:

- am Fristtag
- 1 Tag vorher
- 3 Tage vorher
- 7 Tage vorher
- individuell

## 9. Antwort-Vorschläge

DocPilot kann Antwortschreiben generieren:

- Widerspruch
- Nachfrage
- Beschwerde
- Antrag
- Kündigung
- Ratenzahlungsanfrage
- Terminverschiebung
- Zahlungsaufschub
- fehlende Unterlagen nachreichen
- formelle Behördenantwort

## 10. DokuForge-Modul

DokuForge kann als Generierungsmodul integriert werden für:

- Rechnungen
- Anträge
- Widersprüche
- Vollmachten
- Anschreiben
- Kündigungen
- Fallchronologien
- Zahlungsvereinbarungen
- Zusammenfassungen
- Nachweisdokumente
